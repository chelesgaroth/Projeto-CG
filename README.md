# Projeto de Computação Gráfica

```
$ cd build
$ mkdir 3dFiles (in case it doesn't exist)
$ cmake ..
$ make
```
* Run Generator: -help (to see commands)
```bash
./gerador -help
```
* Run Engine:
```bash
./motor sistemaSolar.xml
```
